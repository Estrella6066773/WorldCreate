# 灵界至高存在

## 概述
- 世界中有几位更为宏大的存在，居于灵界（以太界的别称）的深远之处。
- 无人能寻得它们本身，只能通过它们在万物中留下的痕迹来察觉其影响。
- 水火土风四大元素便源自这四位存在，构成世界运作的根基。

## 四位存在

| 存在 | 起源/性质 | 对应元素 |
|------|-----------|----------|
| **光** | 由“神”所创造 | 火 |
| **暗** | 从光上褪下的 | 水 |
| **天空** | 隔绝洪水的 | 风 |
| **大地** | 被认为是巨鱼巴哈姆特 | 土 |

- **光**：神最初创造之光，其痕迹化生为火之元素。
- **暗**：自光上褪落而成，与暗对应的是水。
- **天空**：承担隔绝洪水之责，对应风（空气）。
- **大地**：常被认作巨鱼巴哈姆特，对应土之元素。

## 特征
- **居所**：皆在灵界（以太界）的深远之处。
- **可知性**：本体不可被寻获，只能借由世间万物的痕迹感知其存在。
- **与元素**：四大元素（火、水、风、土）分别源自这四位存在。

## 与圣经创造的对应
- 对应圣经中神在创世之初的创造：**光**、**暗（水）**、**空气（天空）**、**大地**。
- 本世界观将四位宏大存在置于灵界深处，并以元素与万物中的痕迹呈现这一创世结构。
