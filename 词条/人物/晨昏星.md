# 晨昏星

## 概述

晨昏星是**路西法**在灵界侧与反抗叙事中被指称的名号之一，强调其曾为「受光者」、却居于光暗交界之处——既像拂晓前最亮的那颗星，也像黄昏将尽时仍悬于天际的那一颗。在现行改写口径中，荒原上的 **[昏昼](昏昼.md)** 被视为路西法以「晨昏星」身份在现世投下的一个**面相**（显现方式），而非与路西法毫无关系的凡人偶然崛起。

## 命名与象征

- **晨与昏**：同时指向「自暗向明」与「自明入暗」的转折；与昏昼自称「以昏暗挽来白昼者」、技艺「纳光为火」在意象上相叠。
- **星**：在教会正典中，路西法叛变叙事常与「自天坠地」相连；「星」既可读作陨落前的显赫，也可读作仍在天幕上被辨认、却已被改写的位置。
- **与异端叙事的衔接**：执火派不承认现世伊勒正统，转而追随路西法；地面线上昏昼所凝聚的「投奔魔鬼者」群体，在神学争议中被指认为魔鬼一侧——「晨昏星」为这一套对立提供了可被引用的灵界侧名称。

## 与路西法的关系

- 路西法在 **[伊勒](../概念/伊勒.md#路西法陨落教会正典叙事)** 正典中的身份是「众天使的领袖」「受光者」，叛变后被放逐至灵界，在城内叙事中沦为恶魔之王。
- 「晨昏星」并非教会册封的正式圣号，更接近**反抗者一侧或灵界记述中的别称**，用于概括其光暗交织、难以被单一阵营收编的性质。
- 是否还存在其他以「晨昏星」为轴的面相或显现，**待与主线统一**，当前明确写出的地面面相为昏昼。

## 与昏昼的关系

- **昏昼**是执火军领袖、大执火者，以「纳光为火」为核心技艺；在新版 **[地面线](../../分支时间线/地面线.md)** 的说明中，昏昼被明确为路西法作为「晨昏星」的一个面相。
- 二者关系可理解为：**晨昏星** = 灵界/叙事层上的名号；**昏昼** = 该名号在灾变—破晓时代于荒原与城市冲突中的具体显现之一。

## 相关群体与组织

- **[执火者](../阵营/执火者.md)**：思想史上与追随路西法的执火派同源脉络；昏昼所领导的武装与现实动员常与这一异端叙事交织。
- **[执火军](../组织/执火军.md)**：由昏昼与织星者等实际组织起来的佣兵力量；神学标签上的「晨昏星」并不等同于执火军编制本身。
