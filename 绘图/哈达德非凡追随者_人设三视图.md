## 设定要点（绘图前须知）

- **双角色**：马杜克（风暴使者/云裂）与阿娜特（丰饶使者/穗歌），哈达德亲选的两位非凡从者。
- **人种**：二人均为**小亚细亚人**——马杜克肤色橄榄色/小麦色，阿娜特为**古铜色**；五官立体，深色毛发，具有地中海—近东地域特征。
- **阿娜特体型**：**身高约一米四的少女**，体态娇小，头身比略大，四肢纤细，胸部与臀部曲线平缓，整体呈现**少女/幼女**的稚气感，而非成年女性。
- **时代与风格**：信仰变革期、破晓时代；服饰与饰品融合古代近东、地中海与荒原游牧风格。

## 一、阿娜特（Anat）— 丰饶使者 / 穗歌

### 1.1 核心设定摘要

- **身份**：晓穗语歌者（穗歌）；马杜克之妹兼配偶；有翼童贞少女神。
- **能力**：丰饶与生命力；武艺与征战；**最喜爱弓箭**，亦擅他兵。
- **性格**：好战嗜血、暴烈与洁净并置；战后以紫螺妆点、竖琴贴胸，吟唱对主君之爱。

### 1.2 人种与体型（必遵）

| 项目 | 描述 |
|------|------|
| **人种** | 小亚细亚人：**古铜色**肤色，深邃眼窝，挺直鼻梁，饱满嘴唇，深色眉毛 |
| **身高** | **约 1.4 米**，明显矮于成年女性 |
| **体型** | 少女/幼女体态：头身比约 1:5.5～1:6，四肢纤细，肩窄，腰细，胸部与臀部曲线平缓 |
| **年龄感** | 约 12～14 岁外观，稚气、灵动，非成熟妩媚 |

### 1.3 面部与头部

- **肤色**：**古铜色**，健康光泽。
- **眼睛**：大而呈杏仁状，**金黄色或琥珀色虹膜**，瞳孔深邃，可带细微裂缝状纹理；眼线细长上挑；浓密深色睫毛。
- **面部彩绘**：眼睛下方、颧骨处有**金色点状与线条状彩绘**；额头中央可有小金色装饰；泪痕状金色条纹自内眼角延伸（仪式/征战用）。
- **面部装饰**：面部有**面纱形状的金砂点缀**（金砂呈面纱状，非实体面纱）。
- **发型**：**乌黑或深棕色长发**，及腰，直发或微卷，中分或自然披散。
- **头饰**：**金色月桂或橄榄枝花环**，叶片精细雕刻；可垂挂细小金色坠饰或浅绿色水滴形宝石；脑后可设**放射状金色尖刺光环**（神圣/少女神意象）。

### 1.4 服饰与配饰

- **上身**：**清凉暴露的白色祭司服饰**，**无袖**，设计简洁，带有明显的宗教或祭司风格；领口有**金色镶边**；**露腰**，露肩或露背，体现活力感。身材娇小与服饰形成反差。
- **臂饰**：双臂佩戴**金色臂环、手镯**；上臂可缠轻薄米色/金色布料，穿插小金珠。
- **腰饰**：**金色腰带**，腰带正中央镶嵌一个醒目的**骷髅头装饰**，与整体的神圣感形成强烈的暗黑反差。
- **下身**：同色系的**白色短裙**，充分露出腿部，体现活力。
- **足部**：**赤脚**。
- **项链**：多层金色项链——细珠链、双股珠链、白色球形珠、**圆形大奖章**（花卉/玫瑰图案）、中央大吊坠镶**红宝石**，辅以蓝/白小珠；肩部珠链自颈延伸至肩。

### 1.5 武器与装备

- **主武器**：**等身木质长弓**（相对 1.4 米身高，约等身长）+ **箭筒**（深色，箭矢银头）；弓弦可见。
- **副武器**：**腰间金叶木杖**，系于腰间，用于钝器或施法；可持杖、其他近战兵器；战后场景可持**竖琴**贴胸。
- **翅膀**：设定为「有翼童贞少女神」——**虚化尖刺状羽翼**，与尖刺光环风格相似（尖刺状、半透明）；与丰饶/神圣意象结合。

### 1.6 姿态与氛围

- **战斗态**：背弓持箭，目光锐利，身体紧绷，蓄势待发。
- **仪式态**：紫螺妆点、竖琴贴胸，表情沉静或略带笑意，暴烈与洁净并置。
- **整体**：稚气与杀意并存；娇小身形与强大战力形成反差；身材娇小与宽松长袍形成反差。

### 1.7 三视图分镜提示

| 视角 | 重点 |
|------|------|
| **正面** | 全身立姿，头饰、项链、清凉暴露无袖白色祭司服、金色腰带骷髅头、弓与箭筒、腰间金叶木杖完整可见；短裙；面纱状金砂点缀；面部彩绘与金瞳；赤脚；虚化尖刺状羽翼；少女体型比例明确 |
| **侧面** | 等身长弓与箭筒背挂方式；腰间金叶木杖；虚化尖刺状羽翼侧视形态；头饰与尖刺光环的侧面轮廓 |
| **背面** | 箭筒、弓背、尖刺状羽翼背面；长发披散；腰带与裙摆背面；赤脚 |

## 二、马杜克（Marduk）— 风暴使者 / 云裂

### 2.1 核心设定摘要

- **身份**：流云碎裂者（云裂）；哈达德从者中掌握**隐秘知识**最多者。
- **能力**：风暴与闪电；谋略与决断；理解事物发展规律。
- **分工**：斗法中**主谋略**，后方判断时机与代价，识破巴力节律与弱点。

### 2.2 人种与体型

| 项目 | 描述 |
|------|------|
| **人种** | 小亚细亚人：橄榄色/小麦色肤色，深邃眼窝，挺直鼻梁，轮廓分明 |
| **体型** | 成年男性，智者/决断者气质；身形精干或魁梧，可据叙事需要调整 |
| **年龄感** | 成熟男性，约 25～40 岁外观 |

### 2.3 面部与头部

- **肤色**：橄榄色或小麦色，健康光泽。
- **眼睛**：**灰蓝色虹膜，如乌云般深邃**；目光锐利，带有智者的审视感。
- **发型**：**浓密的深黑色头发**，长度及肩，呈现**自然的大波浪卷曲形态**，蓬松而富有层次感；整体带有随性不羁的感觉，又不失庄重。
- **头饰**：额前佩戴**带云纹浮雕的银箍**，将大部分刘海向后束起，露出饱满的额头和清晰的面部轮廓；脑后用同色系发带或皮筋束成**很小的低马尾**，部分卷曲发尾自然垂落颈后和肩背。
- **面部细节**：两侧鬓角和部分发丝自然垂落脸颊两侧，与**胡须**轮廓相呼应，增添硬朗气质。
- **光环**：脑后**雷纹灰蓝色光环**，闪电纹样，与风暴意象呼应。

### 2.4 服饰与配饰

- **上身**：深色长袍或战甲；肩部可有**风暴/闪电纹饰**；布料或皮革，带有游牧与城邦融合感。
- **配饰**：**银质护手**；**腰带悬挂大量绘有神秘纹路的方形小陶片**；可另挂符咒、卷轴或象征「隐秘知识」的小物。
- **色彩**：深蓝、灰蓝、青铜、黑——与风暴、雷电、高天意象呼应。

### 2.5 武器与装备

- **非主战**：马杜克主谋略，非正面兵戈；可持**权杖、卷轴或罗盘**类道具，象征规律与预判。
- **风暴意象**：周身可有**细微闪电纹路**或**流云状以太光效**；背景可现风暴、乌云、碎裂云层。

### 2.6 姿态与氛围

- **智者态**：站立或端坐，目光远眺，手持卷轴或罗盘，神情沉静而锐利。
- **决断态**：手势为指向或裁决，与后方指挥、时机判断呼应。
- **整体**：威严、冷静、谋略感；与阿娜特的暴烈形成对照。

### 2.7 三视图分镜提示

| 视角 | 重点 |
|------|------|
| **正面** | 全身立姿，云纹银箍、雷纹灰蓝光环、服饰纹样、腰带方形小陶片、权杖/卷轴；饱满额头、鬓角垂发、胡须；智者气质与风暴意象 |
| **侧面** | 银箍与光环侧面轮廓；很小的低马尾束发；服饰层次；道具持握方式 |
| **背面** | 很小的低马尾背面形态，卷曲发丝自束带处散开；长袍/披风背面；雷纹灰蓝光环；风暴或流云纹饰 |

### 组合绘图提示

- **双人立绘**：马杜克略后或略高，阿娜特略前或略低；形成「谋略—兵戈」「后方—前方」的空间关系。
- **兄妹/配偶**：可保留亲密但不逾矩的肢体语言（如并肩、牵手），与童贞少女神叙事兼容。
