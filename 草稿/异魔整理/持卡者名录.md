## 持卡者名录

### 女巫之主
- 𝕬𝖑𝖛𝖊𝖗𝖙𝖆 阿尔维塔 (Alverta)
- 持有卡牌：魔术师 / 星星 / 高塔
- 恶念
- 背景身份：魔女，高塔议庭首席

### 银月圣女
- 𝕬𝖒𝖆𝖗𝖎𝖘 阿玛瑞斯 (Amaris)
- 持有卡牌：女祭司 / 月亮 / 节制
- 自我约束
- 性格外貌：银发、蓝眸金瞳、小女孩、身上有改造痕迹（月民风格）
- 改造：手部/腿部，右侧脸部存在圆圈状纹路（月民风格：有蓝银色电路纹路，没有机械关节）
- 服装：纯白的宽大祭司长袍，长袍的边缘镶金（尺寸偏大）
- 身份背景：月民遗脉，月城城主，炼金术守密者

### 夜之女皇
- 𝓥𝓾𝓵𝓬𝓪𝓷𝓾𝓼 瓦肯努斯 (Vulcanus)
- 持有卡牌：女皇 / 世界 / 倒吊人
- 背叛的种子
- 身份背景：魔域新掌控者，实际上在抑制异魔爆发

### 海帝皇
- 𝑽𝒊𝒄 𝑨𝒓𝒔𝒉𝒂𝒗𝒊𝒏 维克·阿尔沙文 (Vic Arshavin)
- 持有卡牌：皇帝 / 战车 / 恶魔
- 被引诱的想法
- 身份背景：正统潮民，篡位者

### 曙光女神
- 𝓐𝓾𝓻𝓸𝓻𝓪 欧洛拉 (Aurora)
- 持有卡牌：教皇 / 太阳 / 审判
- 一份满怀仇恨的执念
- 身份背景：曙光神教开创者，曙光女神本尊

### 塑命女妖
- 𝕮𝖍𝖊𝖘𝖘𝖎𝖆 切希娅 (Chessia)
- 持有卡牌：隐者 / 命运之轮 / 正义
- 内心对公正的追求
- 身份背景：隐修会副会长，将“愚者卡”投入世界的人

### “剑锋”
- 𝕾𝖊𝖗𝖆 希拉 (Sera)
- 持有卡牌：死神 / 力量 / 恋人
- 一份满怀期盼的执念
