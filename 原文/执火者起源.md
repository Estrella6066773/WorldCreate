# 执火者起源

执火者最初发源于一个部族。在灾变时代（大洪水爆发后的百年内），任何存在都可能获得奇异的力量，但这也使得它们陷入混沌，不再是他们自己。大洪水带来的异变像辐射改变基因一样扭曲着一切事物，使得大地开始思考，植物开始觅食，让不受保护的人类变得愈发不像人类。在这期间，有的人幸运地没被异变夺去生为人的资格，还像良性突变一样获得了特别的能力。

而昏昼，就是这么一个在荒原部落中崛起的强者。他的部族因为逃离城市而被城市人追猎至被黑暗遮蔽的食黯森林。这里的植被和土地发生了异变，黑叶的巨木遮天蔽日，而黑色的土地化成的邪物会在阴影中悄悄将人吞噬。部落中人为了躲避猎人们，只敢借着微弱的烛火前进。

在此期间，有一位少年与部落的众人分散。他跌进阴影，脚下的黑土像在蠕动——再往前一步，就会成为食黯森林的养料。他咬牙擦亮手中的烛火。

微光一起，他看见地上那些凋敝的枯草。本以为是死物，却在烛焰靠近时微微蜷缩，像在躲避，又像在渴求。少年把烛火放低，让光落在草叶上；那些常年不见天日的枯草竟在光照下静了下来，不再朝他的脚踝攀缠。他屏住呼吸，忽然明白了：这森林之所以吞噬黑暗中的人，是因为光被头顶的黑叶巨木夺尽了；得不到阳光的野草被异变逼出了另一种活法——从活物身上夺取养料。而光，能让它们重新安静。那树本不是天生遮天蔽日的树种，是异变把黑叶强加给它们；高大的夺走了光，矮小的便在绝境里学会了吃人。可野草并没有忘记光；它们只是等着，等能再次在阳光下沐浴的那一天。

——那么，若光已被夺走，能做的就不是祈求，而是把仅剩的光收进自己身体里，再变成火。

念头落下的那一刻，他掌中的烛焰微微一颤。烛光正一点点变暗，仿佛被什么吸走了，而烛芯却愈发灼人。他感到某种东西从四周的光里渗进皮肤，在体内汇聚，再顺着指尖涌向那团火。火更炽了，光却先暗后亮——先被夺走，再被还回来。以太的恩赐在那一刻降临：他冲破了「只能依赖外界之光」的桎梏，成了能拿光养火、以火生光的人。

而这位能将光变化为火的少年，就是未来那推翻高耸城墙、将节点网络撕成碎片的大执火者：昏昼。

2. 大执火者的「大」为 great，这个前缀在他所在的时代只代指他一人。
